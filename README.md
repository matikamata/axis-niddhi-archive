# 🏛️ AXIS-NIDDHI — PureDhamma Static Archive

A static, offline-first preservation build of PureDhamma content.

- 748 teachings
- PT-BR + EN
- Offline search
- Pāli glossary (791 terms)
- 481 audio files
- Deterministic build (reproducible)

## Purpose

This project exists for preservation, portability and offline access.
It does not modify the original teachings.

## Current Status

Work in progress.
Some audio files are still being restored.
Service Worker (PWA mode) will be activated in a future release.

## Contribute

- Record missing Pāli pronunciations
- Help restore missing audio files
- Test on mobile and offline scenarios
- Open issues for broken links or inconsistencies
